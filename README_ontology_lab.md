
# Ontology Lab (Prototype)

This prototype shifts the experiment away from graphics.

Instead of evolving pixels, it evolves rewrite rules.

Current capabilities:
- Rewrite operators reproduce and mutate.
- Low-performing operators go extinct.
- High-performing operators leave descendants.
- The output is evolutionary statistics rather than images.

What it does **not** yet do:
- Invent entirely new symbols.
- Rewrite the rewrite engine itself.
- Develop a semantic layer.
- Construct geometry from relational history.

Those are the next research milestones.
